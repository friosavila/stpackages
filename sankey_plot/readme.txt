TITLE
      'sankey_plot': Module to create Sankey type plots

DESCRIPTION/AUTHOR(S)
      
	'sankey_plot' is a command that can be used to create Sankey type plots. 
	Colors and pstyle's can be defined using external variables. But can also be controlled using colorpalette by Ben Jann.
	the package comes with some dataexamples to try.
	
      KW: sankey plot
      KW: alluvial plot 
      KW: stream graph

      Requires: Stata version 16, 
      Also:     palettes, colrspace
