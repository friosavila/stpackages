TITLE
      'KY_FIT': Modules for Fitting mixture models of the Kapteyn-Ypma type to linked survey and administrative data

DESCRIPTION/AUTHOR(S)
      
    'ky_fit' is a set of rutines focus on estimating finite mixture models of the Kapteyn-Ypma type to analyze linked survey and administrative data on labour earnings (or similar variables), taking account of various types of measurement error in each data source. Different combinations of error-ridden and/or error-free observations characterize latent classes. This command can be used to fit a general class of finite mixture models. It also provides post-estimation commands for assessment of reliability, marginal effects, data simulation, and prediction of hybrid earnings variables that combine information from both data sources.
