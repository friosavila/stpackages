TITLE
      'joy_plot': Module to create joyplots and violin plots.

DESCRIPTION/AUTHOR(S)
      
	'joy_plot' is a command that can be used to create joyplots, also known as ridgeplots.
	These are plots for a single variable across different categories. It also allows you to easily use or change colors with colorpalette options (if installed).

      KW: joyplot
      KW: ridgeplot
      KW: kdensity
      KW: violin plot

      Requires: Stata version 14, 
      Also:     palettes, colrspace
