TITLE
      'waffle_plot': Module to create Waffle plots

DESCRIPTION/AUTHOR(S)
      
	'waffle_plot' is a command that can be used to create waffle plots. It is a wrapper for scatter plots.
	These plots are an alternative to pie graphs, and can be used to represent structure of data.
	
      KW: waffle
      KW: waffle plot
      KW: scatter 

      Requires: Stata version 16
